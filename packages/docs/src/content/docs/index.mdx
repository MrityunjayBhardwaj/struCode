## What Stave is

Stave is a browser IDE for **live-coded music and visuals**. It runs four runtimes in one workspace — patterns (Strudel, Sonic Pi) and graphics (p5.js, Hydra) — so a piece of music and the visual that reacts to it can live in adjacent tabs and talk to each other through a shared event stream.

Everything runs locally. Projects are stored in your browser (IndexedDB + CRDT), and a share link is a fragment URL you can drop into chat.
