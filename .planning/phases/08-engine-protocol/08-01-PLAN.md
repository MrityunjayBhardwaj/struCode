# Plan 08-01: Engine Interface, StrudelEngine Conformance, LiveCodingEditor

## Objective
Define the LiveCodingEngine interface and EngineComponents type system. Make StrudelEngine implement it via a `components` getter. Extract LiveCodingEditor as an engine-agnostic React component. Reduce StrudelEditor to a thin wrapper that instantiates StrudelEngine and passes it to LiveCodingEditor.

## Context
StrudelEditor currently hard-imports StrudelEngine, instantiates it internally, and calls 12 methods directly. This plan decouples the editor from any specific engine by introducing a generic interface and splitting the editor into two layers.

Key invariants from project catalogues:
- UV2 (Framework Prototype Sovereignty): Strudel monkey-patching stays inside StrudelEngine
- UV4 (Async Construction): init() must complete before evaluate()
- PK1 (StrudelEngine.evaluate lifecycle): The capture chain stays internal to StrudelEngine
- New: Component Bag Optionality -- consumers MUST check existence before access
- New: Engine Lifecycle Contract -- init -> evaluate -> play -> dispose ordering
- New: Backward Compatibility -- StrudelEditor wrapping LiveCodingEditor must produce identical behavior

Error patterns to watch:
- U1 (Timing): LiveCodingEditor must await init() before evaluate(), same as current StrudelEditor
- U4 (Mock Divergence): Conformance tests must test the real interface shape, not mock it

       get components(): Partial<EngineComponents> {
         const bag: Partial<EngineComponents> = {
           streaming: { hapStream: this.hapStream },
         }
         if (this.analyserNode && this.audioCtx) {
           bag.audio = { analyser: this.analyserNode, audioCtx: this.audioCtx }
         }
         bag.queryable = {
           scheduler: this.getPatternScheduler(),
           trackSchedulers: this.trackSchedulers,
         }
         // Build inlineViz from vizRequests + line scanning
         if (this.vizRequests.size > 0 && this.lastEvaluatedCode) {
           bag.inlineViz = {
             vizRequests: this.buildVizRequestsWithLines(this.vizRequests, this.lastEvaluatedCode),
           }
         }
         return bag
       }
