Define SynthBackend interface for per-track audio routing. Implement for SuperSonic. Wire per-track AnalyserNodes through the adapter to inline viz zones. Real oscilloscope per-track.

# Tasks

- viewZones: use per-track analyser when available

- Expose per-track analysers in components.audio.trackAnalysers

- When trackAnalysers has an entry for this trackKey, pass it as the zone's analyser
